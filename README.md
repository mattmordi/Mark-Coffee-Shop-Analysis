## Overview
This project provides an analysis of coffee shop trends, customer preferences, and sales data.

## File Structure
- **data/**: Contains datasets used for analysis.
- **notebooks/**: Jupyter notebooks with analysis and visualizations.
- **scripts/**: Python scripts for data processing.
- **results/**: Generated reports and visualizations.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/mattmordi/Mark-Coffee-Shop-Analysis.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Mark-Coffee-Shop-Analysis
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```
